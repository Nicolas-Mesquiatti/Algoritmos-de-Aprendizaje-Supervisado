# Algoritmos de Aprendizaje Supervisado en Python

Este repositorio reúne tres implementaciones fundamentales de algoritmos clásicos de Machine Learning, desarrolladas desde cero con fines educativos.

##  Algoritmos incluidos
1. **FOIL (First Order Inductive Learner)**  
   - Inducción de reglas lógicas y cálculo del *FOIL Gain*  
   - Ejemplos prácticos y verificación manual de resultados

2. **Clasificador Bayesiano**  
   - Implementación de Naive Bayes para datos categóricos  
   - Ejemplo con conjuntos de datos pequeños y visualización de probabilidades

3. **Árbol de Decisión (ID3 / C4.5 simplificado)**  
   - Cálculo de entropía, ganancia de información y construcción del árbol  
   - Ejemplo ilustrativo paso a paso

##  Tecnologías utilizadas
- Python 3.x  
- Numpy / Pandas (opcional)  
- Matplotlib (para visualización)

##  Objetivo
Proyecto educativo desarrollado para practicar la implementación desde cero de algoritmos de IA, comprensión de la teoría y validación manual de resultados.
